# Flying Food

Anna Tsirbas & Pippin Barr

A game to help aid Mr.Frog in search for food. Clicking your mouse will help time Mr.Frog's tongue, to catch delcious flies. There is a time amount before Mr.Frog starves to death!

[Play it!](https://7antics.github.io/cart-253/Jams/Mod_Jam/frogfrogfrog/)
